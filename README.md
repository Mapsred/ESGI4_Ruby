RubyPlanet
======

Requirements
-----------------

* Ruby **2.5.1**
* Rails **5.2.0**

Installation
-----------------

```bash
git clone git@github.com:Mapsred/RubyPlanet.git
cd RubyPlanet
bundle install
bin/rails db:migrate RAILS_ENV=development

bin/rails server -b 0.0.0.0 -p 3000 -e development
```
